# Sito Termoidraulica Lauri

Nuovo progetto web creato da zero.

## Obiettivo
Realizzare un sito moderno, chiaro e orientato alla conversione per Termoidraulica Lauri.

## Priorità del progetto
- aumentare richieste di preventivo e contatti
- valorizzare showroom, consulenza e progettazione bagno
- distinguere percorsi per privati e professionisti
- rendere il sito più ordinato, veloce e facilmente espandibile
